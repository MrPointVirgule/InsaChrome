A propos

INSAChrome est une extension sous licence Apache V2 permettant aux étudiants de l'INSA de Lyon de se connecter rapidement aux services web proposés par l'INSA.

InsaChrome utilise dans sa popup un fond d'écran de Subtle Patterns sous licence CC BY-SA 3.0

Le projet est disponible sur https://github.com/MrPointVirgule/InsaChrome , si vous souhaitez voir le code / y contribuer grâce à un fork.

Les B5 rachètent.

Si vous avez des reflexions ou des idées d'améliorations, vous pouvez m'en faire part par mail (paul-louis.hery@insa-lyon.fr), ou en vrai.
